### Agenda
### <strong>1. Deep Dive: Bitcoin Scripts</strong>
- [Mastering Bitcoin: Transactions & Scripts](https://github.com/bitcoinbook/bitcoinbook/blob/develop/ch06_transactions.adoc)
  - **P2PK** — Pay to Public Key
  - **P2PKH** — Pay to Public Key Hash
  - **P2SH** — Pay to Script Hash
  - **P2WPKH** — Pay to Witness Public Key Hash (SegWit)
  - **P2WSH** — Pay to Witness Script Hash (SegWit)

### <strong>Hands On Testing: How Each Script Works</strong>
 Deep dive into hands on testing to understand how each Bitcoin script type works in practice,constructing, signing, and broadcasting transactions for each script type and observing the differences in locking/unlocking conditions, witness data, and transaction weight.
